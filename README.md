## Installation ##

```
$ go get -u github.com/TibebeJs/yenepay.sdk.go
```

Alternatively, If you are using go-mod to manage your project, simply add yenepay to your `go.mod` file:

```
module github.com/x/y

go 1.15

require (
    github.com/TibebeJs/yenepay.sdk.go
)
```

And simply import in your source code.

## Testing ##
To run the tests, use the following command:
```
$ go test -v ./...
```

Alternatively, you may use [`goconvey`](https://github.com/smartystreets/goconvey) (Recommended)

To install [`goconvey`](https://github.com/smartystreets/goconvey), run:
```
$ go get github.com/smartystreets/goconvey
```

After installation, use the following to execute the tests:
```
$ goconvey
```
